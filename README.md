# MineSweeper GUI

A classic Minesweeper game developed in Python using Tkinter.

## Features

- Graphical user interface
- Three difficulty levels (8×8, 12×12, 16×16)
- Random mine generation
- Recursive opening of empty cells
- Automatic adjacent mine counting
- Restart functionality
- Responsive board scaling
- Win/Lose detection

## Planned Features

- Safe first click
- Flag system
- Timer
- Mine counter
- Improved user interface
- High score system

## Requirements

- Python 3.x
- Tkinter

## Installation

```bash
git clone https://github.com/yourusername/MineSweeper-GUI.git
cd MineSweeper-GUI
python3 main.py
```
